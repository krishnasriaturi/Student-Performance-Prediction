<h1><b>Student Performance Prediction</b></h1>

<p>
<u><b><span style="color:#2E86C1;">Overview</span></b></u><br>
This project predicts student academic performance using a hybrid machine learning approach that combines deep learning with optimization techniques. A Convolutional Neural Network (CNN) is integrated with the Butterfly Optimization Algorithm (BOA) to enhance model performance and predictive accuracy on structured educational data. The system classifies outcomes into Fail, Withdrawn, Pass, and Distinction.
</p>

<p>
<u><b><span style="color:#2E86C1;">Technology Stack</span></b></u><br>
- Python <br>
- TensorFlow / Keras <br>
- Pandas and NumPy <br>
- Streamlit <br>
- Butterfly Optimization Algorithm (BOA)  
</p>

<p>
<u><b><span style="color:#2E86C1;">Key Features</span></b></u><br>
- Hybrid CNN + Butterfly Optimization based prediction model <br>
- Improved accuracy through metaheuristic optimization <br>
- Predicts student performance with confidence scores <br>
- Interactive web interface using Streamlit <br>
- File upload functionality for dataset and model <br>
- Visualization of prediction probabilities  
</p>

<p>
<u><b><span style="color:#2E86C1;">Model Approach</span></b></u><br>
- Data preprocessing using normalization techniques <br>
- Feature transformation for CNN compatibility <br>
- CNN used for classification of student outcomes <br>
- Butterfly Optimization Algorithm applied for parameter tuning and performance enhancement <br>
- Evaluation based on prediction confidence and class distribution  
</p>

How to Run  

1. Clone the repository  
2. Install dependencies:  
   pip install -r requirements.txt  
3. Run the application:  
   streamlit run App.py  
4. Upload the dataset and model using the sidebar  

Notes  
- The dataset and trained model are not included in the repository  
- Ensure both files are uploaded before running predictions
